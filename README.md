**Quazarquest - DES-315 Gunship Controls Project**

A tech demo featuring a modular top-down gunship controller and emitter system for projectiles and VFX.
Automatically records telemetry data about the various waves and ships.

**Controls**
- WASD / Arrow Keys: Tank controls to steer the ship
- . / , or Number Keys: Switch ship
- LMB / Spacebar: Fire Main weapon
- RMB / Enter: Fire Special weapon
- Ctrl + + / Scroll Up: Zoom in
- Ctrl + - / Scroll Down: Zoom out
- L: Toggle debug drawing lines for the player
- Ctrl + A: Toggle autoplay for the player
- R: Reset game
